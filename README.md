# Corrector Ortográfico

Aplicación de escritorio desarrollada en Python con Tkinter y LanguageTool.

## Funciones

- Corrección ortográfica
- Corrección gramatical
- Interfaz gráfica
- Resaltado de errores
- Contador de errores

## Requisitos

- Windows
- Java JDK 17 o superior

## Uso

1. Ejecutar `CorrectorOrtografico.exe`
2. Escribir el texto
3. Presionar el botón `Corregir`
4. Revisar el texto corregido

## Tecnologías usadas

- Python
- Tkinter
- LanguageTool
- PyInstaller
